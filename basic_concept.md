
#  Ansible learn resources

date: 2016/03/02 by marxwang 

## **1. Ansible 環境**

* _controller_ python > 2.7
* _client _python > 2.5
* _ansible_ (pip/yum/apt) 最新2.0

## 2.學習 Ansible 專注focus 幾件事

* Inventory (hosts) file
* playbook(s)
* role(s)
* variables
